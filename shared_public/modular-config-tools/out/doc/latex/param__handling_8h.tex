Decodes an U\+RI encoded and null terminated string into a standard A\+S\+C\+II string. The input string content is replaced by the decoded string in case of success. In case of an error the original input string is left untouched.

\begin{DoxyWarning}{Warning}
Decoding of non U\+RI encoded strings may result in wrong or unexpected results, in some cases also without detection and signaling of an error/failure.
\end{DoxyWarning}

\begin{DoxyParams}{Parameters}
{\em sz\+Argument} & U\+RI encoded and null terminated input string to decode. String content is replaced in case of success with decoded string content. Buffer/string size left untouched (decoded string is always shorter or equal).\\
